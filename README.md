# legal-envector-mp

Static legal-notices site for **Heaan Inc.** — the U.S. seller of record for
enVector on Google Cloud Marketplace. Hosted via GitHub Pages.

## Pages

- [`/privacy`](https://cryptolabinc.github.io/legal-envector-mp/privacy) — Privacy Policy (English; authoritative)
- [`/privacy.ko`](https://cryptolabinc.github.io/legal-envector-mp/privacy.ko) — 개인정보 처리방침 (Korean; advisory translation)

## Local preview

```sh
gem install bundler jekyll
jekyll serve
# open http://localhost:4000/legal-envector-mp/
```
